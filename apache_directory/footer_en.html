Documentation

You are here because you want to set up an Internet Cube (La Brique Internet, in French) by yourself. If you are interested by a plug-n-play Cube, please contact a non-profit organization (see the map).

An Installation Guide (fr) is available. All information about hardware, YunoHost apps and community channels can be found on the official website.

If you want information about the Debian image itselft, you can read the README of sunxi-debian and build.labriqueinter.net.

Quick Guide

This command will download the latest image and install your SD card (add -2 to the end for a LIME2):

bash <(wget -qO- http://repo.internetcu.be/install-sd.sh)

For the SSH access and the YunoHost post-installation, see the Post-Installation section below.

Quick Guide (encrypted)

For even further privacy, the root file system of your Cube can be encrypted (images encryptedfs).

The hard drive will have to be unlocked each time your Cube is rebooting. You will be able to remotely unlock the hard drive by accessing to your Cube with HTTPS. For that, you will have to use the public IPv4 from your ISP, with a correctly configured port-forwarding on your router (or directly IPv6 if your are lucky).

This command will download the latest image and install your SD card with an encrypted file system (add -2 to the end for a LIME2):

bash <(wget -qO- http://repo.internetcu.be/install-sd.sh) -e

For the SSH access and the YunoHost post-installation, see the Post-Installation section below.

Post-Installation

SSH is available on the port 22:

root:olinux

The first thing to do is upgrading Debian/Yunohost:

% sudo apt-get update && sudo apt-get dist-upgrade

You can now do the YunoHost post-installation. Then, install VPN Client and Wifi Hotspot on YunoHost.

Enjoy!
