Documentation

Vous êtes ici parce que vous voulez configurer vous-même une Brique Internet. Si vous voulez une Brique clé en main, merci de contacter un FAI associatif local (voir la carte).

Un guide d'installation détaillé est disponible. Vous trouverez toutes les informations concernant le matériel supporté et les apps YunoHost sur le site officiel.

Les informations à propos de l'image Debian sont disponibles dans les README de sunxi-debian et build.labriqueinter.net.

Guide express

Cette commande va télécharger la dernière image et l'installer sur votre carte SD (ajouter l'option -2 à la fin pour une LIME2) :

bash <(wget -qO- http://repo.labriqueinter.net/install-sd.sh)

Vous pouvez aussi télécharger le script et utiliser -h pour voir les autres options.

Pour l'accès SSH et la post-installation de YunoHost, voir la section Post-installation ci-dessous.

Guide express (avec chiffrement)

Pour encore plus de protection de vos données, le système de fichiers de votre Brique peut être chiffré (images encryptedfs).

La partition devra être déchiffrée à chaque redémarrage de la Brique. Il sera possible de procéder au déchiffrement à distance, grâce à une interface web accessible en HTTPS. Vous devrez alors utiliser l'adresse IP publique de votre FAI (directement en IPv6, ou en IPv4 avec de la redirection de ports sur votre box).

Cette commande va télécharger la dernière image et l'installer sur votre carte SD, avec un système de fichiers chiffré (ajouter l'option -2 à la fin pour une LIME2) :

bash <(wget -qO- http://repo.labriqueinter.net/install-sd.sh) -e

Vous pouvez aussi télécharger le script et utiliser -h pour voir les autres options.

Pour l'accès SSH et la post-installation de YunoHost, voir la section Post-installation ci-dessous.

Post-installation

SSH est disponible sur le port 22 :

root:olinux

La première chose à faire est de mettre à jour Debian/YunoHost :

% sudo apt-get update && sudo apt-get dist-upgrade

Vous pouvez maintenant faire la post-installation. Puis installer VPN Client et Wifi Hotspot sur YunoHost.
