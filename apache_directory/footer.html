Documentation

Vous êtes ici parce que vous voulez configurer vous même une brique internet. Si vous voulez une brique "clé en main", merci de contacter un FAI associatif local (voir la carte).

Un guide d'installation (fr) détaillé est disponible. Vous trouverez toutes les informations sur le matériel supporté et les apps YunoHost sur le site officiel.

Vous trouverez les informations sur l'image Debian sur le README de sunxi-debian et de build.labriqueinter.net.

Guide Rapide

Remplacer labriqueinternet_latest.img par le nom de l'image téléchargée et NomDuDevice par le nom de votre micro-SD (par ex. /dev/sdb, /dev/mmcblk0) :

% sudo dd if=labriqueinternet_latest.img of=/dev/DEVICENAME bs=1M

Pour l'accès SSH et la post-installation de YunoHost, voir la section Post-Installation ci-dessous.

Guide Rapide (avec chiffrement)

Pour encore plus de protection de vos données, grâce à l'image encryptedfs, le système de fichiers root de votre brique peut-être chiffré.

La partition devra être déchiffrée à chaque redémarage de la brique. Il sera possible de déchiffrer la partition en accédant à une interface web en HTTPS. Vous devrez alors utiliser l'adresse IP publique de votre FAI (directement en IPv6, ou avec de la redirection de ports sur votre routeur en IPv4).

Remplacer labriqueinternet_latest.img par le nom de l'image téléchargée, NomDuDevice par le nom de votre micro-SD (par ex. /dev/sdb, /dev/mmcblk0) et choisir a20lime ou a20lime2 :

% git clone https://github.com/bleuchtang/sunxi-debian && cd sunxi-debian
% sudo bash olinux/create_device.sh -d /dev/DEVICENAME\
-b labriqueinternet_encryptedfs_latest.img -e -u a20lime

For the SSH access and the YunoHost post-installation, see the Post-Installation section below.

Post-Installation

Il y a un serveur SSH sur le port 22 :

root:olinux

Commencez par mettre à jour Debian et YunoHost :

% sudo apt-get update && sudo apt-get dist-upgrade

Vous pouvez maintenant faire la post-installation. Puis installer VPN Client et Wifi Hotspot sur YunoHost.
